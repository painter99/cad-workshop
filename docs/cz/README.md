# Dokumentace a metodika

V této sekci popisuji své postupy, poznámky k frameworku a způsoby, jak zefektivnit CAD modelování.

## AI-Assisted Workflow

Můj přístup k experimentování kombinuje lidský záměr a efektivitu AI:

1.  **Nápad a popis:** Slovně definuji tvar a technické parametry.
2.  **AI Iterace:** LLM připraví základní skript v build123d. Kód je pro AI mnohem srozumitelnější formát než binární soubory GUI editorů.
3.  **Okamžitá validace:** Pomocí **OCP CAD Vieweru** vidím výsledek okamžitě.
4.  **Ladění:** Manuálně doupravím detaily.

## Poznámky
*Zde budu doplňovat konkrétní postřehy k instalaci a používání build123d.*
