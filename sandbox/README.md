# Sandbox

Toto je nejživější část repozitáře. Zde probíhají prvotní experimenty, testování AI generovaného kódu a ladění tvarů.

- Kód zde může být "neuklizený" nebo ve stavu pokus/omyl.
- Soubory se mohou často měnit.
- **Aktuální obsah:** Rychlé skripty pro generování prvních modelů a testování workflow build123d.
